# Document Samples

## Purpose

This folder contains reference documents collected during the research phase of the project.

These documents are used to:

- Understand the structure and format of real police and legal documents.
- Study the investigation workflow followed by law enforcement agencies.
- Design document templates for automated generation.
- Identify common fields required for document creation.
- Assist in documentation and system design.

## Available References

- FIR Samples
- Remand Request
- Medical Examination
- Seizure Memo
- Panchanama
- Investigation & Medicolegal Reference Manuals

## Note

These documents are provided only as reference material for research and development purposes. They are **not** used directly by the application and do not form part of the system's knowledge base or AI retrieval pipeline.
